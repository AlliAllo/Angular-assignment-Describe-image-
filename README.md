# Image Description Using OpenAI

Simple Angular frontend with a Node.js backend that uploads an image and returns an AI-generated description.

## Prerequisites
- Node.js
- npm

## Install
```bash
npm install
```

## API key
Add your own OpenAI API key in a .env file. Path should be: src\backend\.env
```bash
OPENAI_API_KEY={Your_OpenAI_API_Key}
```

## Run (Backend + Frontend)
```bash
npm run dev
```
